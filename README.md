# JS Practice — 100 JavaScript Exercises

An interactive web app with 100 hand-picked JavaScript exercises, from your very first `console.log` to async/await, classes, and mini projects. Built for learners who want to understand JavaScript deeply — not just copy code.

## Features

- 100 exercises organized into 18 chapters
- Progress tracking saved automatically in your browser
- Filter by difficulty (Easy / Medium / Hard) or completion status
- Search exercises by keyword
- Each exercise includes a description, code example, and a hint
- Clean dark-themed interface, works on mobile and desktop
- No installation, no dependencies — pure HTML, CSS, and JavaScript

## How to Use

1. Open the deployed site (or `index.html` locally)
2. Click any exercise card to see the task, an example, and a hint
3. Try to solve it yourself first — type the code, do not copy
4. When stuck, read the hint, then search online before checking the example
5. Click "Mark as Done" once you complete it
6. Your progress is saved automatically and persists between visits

## Recommended Pace

- **5 exercises per day** → finish in 20 days
- **10 exercises per day** → finish in 10 days

Consistency beats speed. Understanding why each line works matters more than finishing fast.

## Running Locally

No build step required. Simply:

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO
```

Then open the HTML file in your browser, or use a local server like the Live Server extension in VS Code.
